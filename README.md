# TikTok Hashtag Analysis (2022,2023,2024,2025)

# Project Overview
This project is a beginner-level data analytics project that analyzes the top 20 trending TikTok hashtags.
The goal is to understand hashtag popularity based on post count and yearly trends using Python.

This project was created as part of my learning journey in data analytics.

## Analysis Performed
- Top 20 trending TikTok hashtags by post count
- Top 20 hashtags in the year 2025
- Data visualization using bar charts
